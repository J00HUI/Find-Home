# HappyHouse, 구해줘 Home
아파트 매매 정보 제공 웹 사이트, 구해줘 Home 🏡
</br>
</br>

## 💡 서비스 소개
[공공데이터포털](https://www.data.go.kr/) API와 [KAKAO Map](https://developers.kakao.com/) API 를 활용하여 아파트 매매 정보 제공과 커뮤니티 기능이 포함된 웹 서비스입니다.
